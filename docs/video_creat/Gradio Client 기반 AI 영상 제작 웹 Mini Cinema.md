# Gradio Client 기반 AI 영상 제작 웹: "Mini Cinema"

**프로젝트 성격:** 저비용/고성능 오픈소스 영상 제작 도구
**핵심 전략:** 내 컴퓨터 사양은 낮아도 OK! 허깅페이스의 강력한 GPU 모델을 **API(Gradio Client)**로 원격 호출하여 사용.

---

## 1. 프로젝트 아키텍처 (어떻게 돌아가나요?)

1. **나의 웹사이트 (Next.js):** 사용자가 글을 쓰는 얼굴(UI) 역할.
2. **두뇌 (Gemini 2.5):** 사용자의 한글을 멋진 영어 프롬프트로 변환.
3. **근육 (Hugging Face Space):** 실제로 영상을 만드는 엔진 (Gradio Client로 연결).
4. **저장소 (JSON):** 만든 영상의 링크를 기억해두는 메모장.

---

## 2. 상세 기능 요구사항 (Step-by-Step)

### **Step 1: 스마트 프롬프트 변환기 (Gemini)**

- **입력:** 사용자가 한글로 "숲속에서 춤추는 요정" 입력.
- **작동:** Gemini 2.5 Flash가 이를 분석해 떡칠된 수식어가 포함된 영어로 변환.
- **출력 예시:** `Cinematic, 4k, a beautiful fairy dancing in a mystical forest, rays of sunlight, high detail...`

### **Step 2: Gradio Client 원격 호출 (핵심!)**

- **방식:** 내 서버에서 직접 AI를 돌리지 않고, 허깅페이스에 이미 떠 있는 `Wan2.1`이나 `CogVideoX` 같은 공간의 주소를 가져와 연결합니다.
- **장점:** 내 컴퓨터는 아무런 힘을 쓰지 않아도 되며, **완전 무료** 혹은 매우 낮은 비용으로 고성능 AI를 쓸 수 있습니다.
- **기술:** `@gradio/client` 라이브러리 사용.

### **Step 3: 씬(Scene) 관리자 (1분 영상용)**

- **구조:** 1분 영상을 만들기 위해 **'장면 1~6'** 번호를 매긴 슬롯을 제공합니다.
- **저장:** 영상 생성이 완료되면 `/data/cache.json`에 `{"scene1": "영상주소", "prompt": "..."}` 형태로 기록합니다.

---

## 3. 초보자를 위한 상세 기술 사양 (코딩용 정보)

### **A. 프론트엔드 (Next.js 15)**

- **컴포넌트 구조:**
    - `InputSection.tsx`: 한글 입력 및 변환 버튼.
    - `VideoPreview.tsx`: 생성된 영상을 보여주는 창.
    - `HistoryList.tsx`: 캐시된 이전 영상 목록 (JSON 데이터 기반).
- **상태 관리:** `useState`를 사용하여 영상 생성 중 로딩 상태(Spinner) 표시.

### **B. API 라우트 (Route Handlers)**

- `api/generate-prompt`: Gemini API를 호출하여 한글 -> 영어 변환.
- `api/generate-video`: Gradio Client를 호출하여 허깅페이스 모델로부터 영상 주소 수신.

---

## 4. UI/UX 디자인 상세 (shadcn/ui 기준)

1. **배경:** 짙은 네이비 배경 (`bg-slate-950`).
2. **입력창:** 하단에 고정된 입력 바 (챗GPT 스타일).
3. **비디오 카드:** * 중앙에 16:9 비율의 비디오 플레이어.
    - 영상 아래에 '다운로드' 버튼과 '장면 저장' 버튼.
4. **사이드바:** 지금까지 만든 장면들을 썸네일로 리스트업 (클릭 시 다시 보기).

---

## 5. 개발자를 위한 "실제 코드 가이드" (맛보기)

나중에 코딩을 시작할 때, AI에게 이렇게 시키시면 됩니다.

> AI에게 줄 프롬프트 예시:
"Next.js 15와 @gradio/client를 사용하여 허깅페이스의 'Wan-Video/Wan2.1-T2V-14B' 모델을 호출하는 API를 만들어줘. 사용자가 텍스트를 보내면 영상 파일 경로를 반환해야 해."
>

## 6. 오픈소스 저작권 및 비용 관리

- **저작권:** `Apache 2.0` 라이선스 모델을 사용하여 상업적 이용 및 배포가 가능하도록 설정.
- **비용:**
    - **Vercel/Gemini:** 무료 티어 범위 내 사용 가능.
    - **Hugging Face:** 타인의 Space를 빌려 쓰는 것은 무료이나 대기 줄이 길 수 있음. (쾌적함을 원할 경우 내 계정으로 복제 후 '시간당 약 800원' 정도의 GPU를 켤 때만 결제)
