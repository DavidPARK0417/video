# 텍스트-투-비디오 모델 선택 가이드

## 현재 사용 중인 모델

### Wan2.1-T2V-14B (권장)
- **Space 주소**: `Wan-Video/Wan2.1-T2V-14B`
- **라이선스**: Apache 2.0 (상업적 이용 가능)
- **특징**:
  - 14B 파라미터로 고품질 영상 생성
  - 1280x720 해상도 지원
  - 최대 10초 영상 생성 가능
  - 무료로 사용 가능 (대기열 있을 수 있음)
- **장점**: 높은 품질, 안정적인 작동
- **단점**: 대기 시간이 길 수 있음

## 다른 무료 오픈소스 모델 옵션

### 1. Stable Video Diffusion
- **제작사**: Stability AI
- **라이선스**: Apache 2.0
- **특징**: 이미지 기반 영상 생성 (텍스트 직접 입력 불가)
- **용도**: 이미지가 있을 때 애니메이션화

### 2. Zeroscope
- **라이선스**: Apache 2.0
- **특징**: 경량 모델, 빠른 생성 속도
- **용도**: 빠른 프로토타이핑

### 3. CogVideoX
- **라이선스**: Apache 2.0
- **특징**: 고품질 영상 생성
- **용도**: 전문적인 영상 제작

## 모델 선택 기준

1. **품질 우선**: Wan2.1 (현재 사용 중) ✅
2. **속도 우선**: Zeroscope
3. **이미지 기반**: Stable Video Diffusion

## 현재 설정

- **모델**: Wan2.1-T2V-14B
- **해상도**: 1280x720
- **영상 길이**: 10초
- **비용**: 무료 (허깅페이스 무료 티어)

## 모델 변경 방법

`app/api/generate-video/route.ts` 파일에서 `AVAILABLE_MODELS` 객체에 새로운 모델을 추가하고, 프론트엔드에서 `modelType` 파라미터를 전달하면 됩니다.

## 주의사항

1. **대기열**: 무료 모델은 전 세계 사용자가 공유하므로 대기 시간이 길 수 있습니다 (1~5분)
2. **API 키**: 일부 모델은 허깅페이스 API 키가 필요할 수 있습니다
3. **파라미터**: 모델마다 파라미터 순서와 형식이 다를 수 있으므로 각 모델의 문서를 확인해야 합니다

## 추천

현재 사용 중인 **Wan2.1-T2V-14B** 모델이 품질과 안정성 면에서 가장 좋은 선택입니다. 
저작권 이슈 없이 무료로 사용할 수 있으며, 상업적 이용도 가능합니다.
