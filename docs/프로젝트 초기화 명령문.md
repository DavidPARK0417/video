2단계: 유튜브 API 및 자막 추출 로직 구현 (API 키 발급 후 입력하세요)
명령문: "이제 백엔드 API 로직을 구현할 거야.

요구사항:

youtube-transcript 라이브러리를 설치해줘.

/app/api/youtube/search/route.ts 엔드포인트를 만들어줘.

이 API는 키워드를 받아서 YouTube Data API v3로 영상을 검색하고, 각 영상의 조회수와 채널의 구독자수를 비교해서 '조회수/구독자수' 비율이 10배 이상인 영상만 필터링해야 해.

검색된 영상들의 ID를 이용해 youtube-transcript로 자막을 가져오는 로직을 포함해줘.

모든 결과 데이터는 /data/cache.json에 저장해서 동일한 키워드 요청 시 API 쿼터를 아낄 수 있게 해줘."
