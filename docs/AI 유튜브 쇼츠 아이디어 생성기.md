# AI 유튜브 쇼츠 아이디어 생성기

## 1. 프로젝트 개요

- **프로젝트명:** Shorts Genius (가칭)
- **목표:** 무엇을 만들지 막막한 초보 유튜버를 위해, 실시간 트렌드와 바이럴 데이터를 분석하여 바로 촬영 가능한 **'제목 + 대본 + 가이드'** 패키지를 제공함.
- **주요 타겟:** 유튜브를 이제 막 시작한 개인 유튜버 (비개발자/입문자)

---

## 2. 핵심 기능 정의

### 기능 1: 트렌드 스카우트 (Trend Scout)

- **목적:** 아무 아이디어가 없는 사용자를 위한 '요즘 뜨는 주제' 추천.
- **주요 동작:**
    - 유튜브 인기 급상승 쇼츠 지표 분석.
    - **기간별 트렌드 비교:** 지난 7일, 30일, 90일 데이터를 비교하여 '떠오르는 주제'와 '식어가는 주제' 구분.
    - **결과물:** 트렌드 키워드, AI 스크립트 초안, 주제별 성공 포맷 태깅.

### 기능 2: 바이럴 벤치마킹 (Viral Benchmarking)

- **목적:** 특정 키워드에서 '대박' 친 영상의 공식 해킹.
- **주요 동작:**
    - 사용자가 입력한 키워드 기반 영상 검색.
    - **고효율 필터링:** 구독자 수 대비 조회수가 10배~1000배 높은 '바이럴' 영상 추출.
    - **채널 규모 필터링:** 나(초보)와 비슷한 규모(예: 구독자 100~1000명)에서 성공한 사례 우선 노출.
    - **심층 분석:** Gemini 2.5 Flash를 통한 영상 자막, 제목 패턴, 업로드 타이밍 분석.
    - **결과물:** 벤치마킹 리포트 (제목 구성, 3초 후킹 포인트, 촬영 가이드).

## 3. 기술 스택 (Tech Stack)

| **구분** | **기술** | **상세 버전/이유** |
| --- | --- | --- |
| **Frontend** | **Next.js 15.5.7** | App Router, 최신 안정 버전 |
| **Language** | **TypeScript** | 코드 안정성 및 AI 코딩 최적화 |
| **Styling** | **Tailwind CSS + shadcn/ui** | 빠른 UI 구현 및 현대적인 디자인 |
| **AI 모델** | **Gemini 2.5 Flash** | 멀티모달(자막/영상) 분석 및 무료 티어 활용 |
| **API** | **YouTube Data API v3** | 공식 수치 데이터(조회수, 구독자 등) 추출 |
| **Lib** | **youtube-transcript** | 유튜브 자막 데이터 추출용 (npm 패키지) |
| **Storage** | **JSON Files** | DB 없이 `/data/cache.json`에 데이터 로컬 저장 |

## 5. 사용자 경험 (UX) 및 화면 구조

1. **Dashboard:** 전체적인 트렌드 요약 리포트 (Rising/Fading 그래프).
2. **Idea Generator:** 버튼 클릭 시 '트렌드 스카우트' 실행 및 카드 형태로 아이디어 나열.
3. **Search & Analysis:** 키워드 검색창 및 바이럴 영상 리스트 (필터 적용 포함).
4. **Result Page:** 선택한 영상의 상세 분석 리포트 및 AI가 써준 최종 제작 가이드.

---

## 6. 개발 로드맵 (단계별 구현)

- **1단계 (환경 구축):** Next.js 15.5.7 프로젝트 생성 및 shadcn/ui 레이아웃 잡기.
- **2단계 (데이터 연동):** 유튜브 API 연결하여 키워드 검색 및 지표(구독자/조회수) 계산 로직 구현.
- **3단계 (AI 분석):** Gemini 2.5 Flash 연결하여 자막 기반 분석 프롬프트 최적화.
- **4단계 (캐싱 및 완성):** JSON 파일 저장 로직 구현 및 전체 UI 다듬기.

---

## 7. 기대 효과

- 초보 유튜버의 기획 시간 80% 단축.
- 데이터에 기반한 기획으로 영상 노출(떡상) 확률 증가.
- 기술적 지식 없이도 AI를 활용한 고수준의 채널 운영 가능.
